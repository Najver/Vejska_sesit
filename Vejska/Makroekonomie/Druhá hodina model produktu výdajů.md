### Spotřeba a úspory

Spotřeba:
Keynsova spotřeba spotřební funkce
C = Ca + c * Yd

C = Spotřeba
Ca = Autonomní spotřeba - nezávisí na disponibilním důchodě
c * Yd = Indukovaná spotřeba - závisí na disponibilním důchodě
Yd = Disponibilní důchod
c = Jak se mi změní spotřeba když se mi změní disponibilní důchod o jednotku
Disponibilní důchod = Část příjmu domácností, která jim zůstane k dispozici poté, co zaplatí daně a povinné odvody, a po započtení sociálních transferů (např. důchody, podpory, dávky)


APC = (C/Yd = Ca + c * Yd) / Yd = **Ca / Yd + C**

APC = Průměrný sklon ke spotřebě
C = Spotřeba 
Yd = Celkový důchod

Růst spotřeby
- Když klesá úroková míra = větší spotřeba (Mají více peněz na utrácení)

Vztah keynesovy spotřeby a úsporové funkce

S = - Sa + sYd

S = Úspory
C = Spotřeba
Sa = Autonomní úspory - nezávislé na disponibilním důchodu

### Rovnovážný produkt ve dvousektorové ekonomice
- Domácnosti a firmy
- **Rovnovážný produkt** = taková úroveň důchodu (HDP), při níž se celkový výstup **rovná celkovým výdajům**

- Y=C+I
- To znamená, že firmy vyrobí přesně tolik, kolik jsou domácnosti a firmy ochotny koupit.
- Není nadprodukce (zásoby by rostly).
- Není podprodukce (zásoby by se vybíraly).

HDP = C + Ip 
IU = 0 
HDP = C + I 
HDP = C + S 
- Protože celý důchod domácností se buď **utratí za spotřebu (C)**, nebo **ušetří (S)**
C + I = C + S 
I = S

PV = C + Ip
PV = HDP
HDP = C + Ip                                                 YD = HDP
HDP = Ca + C * HDP + Ip

HDP - C * HDP = Ca + Ip
(1 - C) * HDP = Ca + Ip
HDP = (1/1-C) * (Ca + Ip)


Ip = Plánované investice
IU = Investiční úniky
I = Investice
PV = Plánované výdaje

Výdajový multiplikátor 
**Výdajový multiplikátor** říká, o kolik se změní **HDP**, když se změní **autonomní výdaje** (například investice nebo vládní výdaje)

- Malá změna výdajů vyvolá větší změnu celkového produktu (HDP)
Delta HDP = (1 / 1 - c) * (delta Ip)

HDP = (1 / 1 - c) * (Ca + Ip)
